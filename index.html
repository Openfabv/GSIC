We build on open tooling — OpenLane, Sky130, the RISC-V ISA — because a fabless studio shouldn't need a seven-figure EDA seat to prove a core in silicon. GISC starts with verified, licensable RISC-V IP and FPGA prototyping services, and reinvests toward custom accelerators for signal processing and edge inference.

01 How a design moves through GISC

A fixed sequence, every time — each stage gates the next.

  1. 01

    RTL Design

    Verilog/SystemVerilog architecture and coding — cores, peripherals, custom instruction extensions.

  2. 02

    Verification

    Verilator + cocotb testbenches, coverage-driven, checked against riscv-arch-test compliance.

  3. 03

    Synthesis & PPA

    Power–performance–area tuning through OpenLane, with static timing analysis for closure.

  4. 04

    FPGA Prototyping

    Xilinx-platform bring-up for real hardware validation before committing to silicon.

  5. 05

    Tapeout

    Shuttle/MPW runs on Sky130 or GF180 — proof of silicon, not just simulation.
